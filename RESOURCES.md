# Resources

## Simulation Assets

| Name | Link | Description |
|------|------|-------------|
| RIGYD | [rigyd.com](https://rigyd.com/) | Converts images / 3D objects / descriptions into simulation-ready assets. Supports Gazebo, Isaac Sim, and more |

| Objaverse | [objaverse.allenai.org](https://objaverse.allenai.org/) | Massive open-source 3D object dataset |
